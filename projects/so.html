Sollaris - the solar system simulator

Project made in C++, allows you to create solar systems and experiment with them. Application also allows to download data about our own solar system from NASA's API, letting you to experiment with current planets. Project was made using C++, integrates with python for easily configurable graphic display and uses curl to connect to api.

Github link: https://github.com/Jan-Wolski/Solar-System
